Sam Cancels a project - GEMS (original) and SwitchJoy.

Samuel (Sparksammy) Lord

There are several reasons I have cancelled these projects, for GEMS:

  • I don't know C worth a hoot.
  • I do, however, somewhat know C#
    • Neither does Oklomsy, the only contributor.
  • Cross compilers don't like third party libraries.
    • Even ones with the same architecture.

And, as for SwitchJoy, the web standards are a moving target, and to maintain it we would have to be working every month until we die on it, or we would have to have several more contributors, and let them work on it.

If you want to continue development on either of these projects, they are on my GitHub, so feel free to fork and improve.

Have a great day!

--Sam Lord
